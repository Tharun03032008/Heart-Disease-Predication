# ❤️ Heart Disease Prediction

## 📌 Description
Heart Disease Prediction is a machine learning project that predicts the possibility of heart disease based on medical data. The system analyzes important health parameters such as age, blood pressure, cholesterol level, and heart rate to determine the risk of heart disease. This project helps in early detection and supports better healthcare decisions.

## 🎯 Objective
The main objective of this project is to build a machine learning model that can predict whether a person is likely to have heart disease. It also provides an easy-to-use interface for users to enter medical details and get prediction results.

## 🛠 Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Plotly

## 📊 Dataset Features
The dataset includes several medical attributes such as:
- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise Induced Angina
- Oldpeak
- ST Slope

## ⚙️ Features of the Project
- User-friendly interface using Streamlit
- Data analysis and visualization
- Heart disease prediction using machine learning
- Interactive charts and graphs
- Simple and easy-to-use system

## 🚀 How to Run the Project
1. Clone the repository
2. Install required libraries
3. Run the Streamlit application
